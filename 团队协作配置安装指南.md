## 🚀 安装步骤

### 1. 安装推荐的VS Code扩展

打开VS Code，会提示安装推荐的扩展。或者手动安装：

#### 后端必需扩展
```bash
# 在VS Code中按 Ctrl+Shift+P，输入：
Extensions: Show Recommended Extensions
```

**核心扩展**:
- ESLint - 代码检查
- Prettier - 代码格式化
- GitLens - Git增强
- Docker - Docker支持
- TODO Tree - TODO管理

#### 前端必需扩展
```bash
# 额外需要安装：
Vue - Official (Volar) - Vue3支持
Stylelint - CSS检查
Tailwind CSS IntelliSense - Tailwind支持
```

### 2. 安装Git Hooks（提交前检查）

#### 后端安装
```bash
cd server/meimei_server

# 安装依赖（如果还未安装）
yarn install

# 安装husky和commitlint
yarn add -D husky @commitlint/cli @commitlint/config-conventional

# 初始化husky
npx husky install

# 添加commit-msg hook
npx husky add .husky/commit-msg 'npx --no-install commitlint --edit "$1"'

# 添加pre-commit hook（lint检查）
npx husky add .husky/pre-commit 'yarn lint'
```

#### 前端安装
```bash
cd server/meimei_ui_vue3

# 安装依赖（如果还未安装）
yarn install

# 安装husky和commitlint
yarn add -D husky @commitlint/cli @commitlint/config-conventional

# 初始化husky
npx husky install

# 添加commit-msg hook
npx husky add .husky/commit-msg 'npx --no-install commitlint --edit "$1"'

# 添加pre-commit hook（lint检查）
npx husky add .husky/pre-commit 'yarn lint'
```

### 3. 配置Git用户信息

```bash
# 全局配置（如果还未配置）
git config --global user.name "你的姓名"
git config --global user.email "你的邮箱"

# 或者为项目单独配置
git config user.name "你的姓名"
git config user.email "你的邮箱"
```

### 4. 验证配置

#### 测试提交规范
```bash
# 正确的提交格式
git commit -m "feat(order): 添加订单列表筛选功能"

# 错误的提交会被拦截
git commit -m "更新代码"  # ❌ 不符合规范
```

#### 测试代码检查
```bash
# 后端
cd server/meimei_server
yarn lint

# 前端
cd server/meimei_ui_vue3
yarn lint
```

## 📝 使用指南

### 提交代码流程

1. **修改代码**
2. **保存文件**（自动格式化）
3. **添加到暂存区**
   ```bash
   git add .
   ```
4. **提交**（会触发检查）
   ```bash
   git commit -m "feat(module): 描述"
   ```
5. **推送**
   ```bash
   git push origin branch-name
   ```

### 交互式提交（可选）

使用 `commitizen` 进行交互式提交：

```bash
# 安装commitizen
yarn add -D commitizen cz-conventional-changelog

# 配置package.json
# 添加: "config": { "commitizen": { "path": "cz-conventional-changelog" } }

# 使用交互式提交
yarn commit
```

### 提交信息格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

**示例**:
```
feat(order): 添加订单批量导入功能

- 支持Excel文件导入
- 支持数据验证
- 显示导入进度

Closes #123
```

## 🔧 常见问题

### Q1: Husky安装失败

```bash
# 清理并重新安装
rm -rf .husky
npx husky install
npx husky add .husky/commit-msg 'npx --no-install commitlint --edit "$1"'
```

### Q2: ESLint检查失败

```bash
# 自动修复
yarn lint --fix

# 如果还有错误，手动修改代码
```

### Q3: 预提交Hook太慢

```bash
# 可以临时跳过（不推荐）
git commit --no-verify -m "message"
```

### Q4: VS Code扩展未生效

1. 重新加载VS Code窗口
2. 检查扩展是否正确安装
3. 查看 `.vscode/settings.json` 是否正确
