## 🚀 使用步骤

### 1. 执行数据库脚本
```bash
mysql -u root -p your_database < database/payment_tables.sql
```

### 2. 在AppModule中导入PaymentModule
```typescript
import { Module } from '@nestjs/common';
import { PaymentModule } from './modules/biz/payment/payment.module';

@Module({
  imports: [
    // ... 其他模块
    PaymentModule,
  ],
})
export class AppModule {}
```

### 3. 配置TypeORM
确保在 `app.module.ts` 或 `ormconfig.ts` 中注册实体：
```typescript
TypeOrmModule.forRoot({
  // ... 其他配置
  entities: [
    // ... 其他实体
    PaymentOrderEntity,
    PaymentChannelEntity,
    DeliverEntity,
  ],
})
```

### 4. 测试API

#### 获取支付页面
```bash
curl http://localhost:3000/checkout/pay?v=ORDER123456
```

#### 创建支付订单
```bash
curl -X POST http://localhost:3000/api/payment/create \
  -H "Content-Type: application/json" \
  -d '{
    "orderNo": "ORDER123456",
    "amount": 180.00,
    "currency": "USD",
    "paymentMethod": "CREDIT_CARD"
  }'
```

#### 查询支付状态
```bash
curl http://localhost:3000/api/payment/query?paymentNo=PAY1234567890
```

## 🔐 安全建议

1. **签名验证**: 所有回调必须验证签名
2. **HTTPS**: 生产环境必须使用HTTPS
3. **密钥管理**: 使用环境变量存储密钥
4. **日志脱敏**: 敏感信息不要记录日志
5. **频率限制**: 添加API频率限制
