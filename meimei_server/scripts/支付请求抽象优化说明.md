# 支付请求参数抽象优化说明

## 📋 优化概述

将支付通道选择逻辑封装到 `PaymentService.createPayment()` 方法中，调用方**无需关心**通道配置细节，只需提供基本支付信息即可。

## ✨ 核心改进

### Before（优化前）

调用方需要手动选择和传入通道信息：

```typescript
// ❌ 复杂且容易出错
const paymentChannel = await this.paymentService.selectCreditCardChannel(
  orderInfo.amount,
  orderInfo.currency,
);

if (!paymentChannel) {
  return { success: false, message: '暂无可用的信用卡支付通道' };
}

const paymentResult = await this.paymentService.createPayment({
  orderNo: checkoutPayDto.v,
  amount: orderInfo.amount,
  currency: orderInfo.currency,
  paymentMethod: 'CREDIT_CARD',
  userId: '',
  cardNumber: checkoutPayDto.number,
  cardExpiry: checkoutPayDto.expiry,
  cardCvv: checkoutPayDto.verification_value,
  cardHolderName: checkoutPayDto.name,
  channelCode: paymentChannel.channelCode,      // ❌ 需要手动传入
  channelNotify: paymentChannel.notifyUrl,      // ❌ 需要手动传入
});
```

### After（优化后）

调用方只需提供基本支付信息：

```typescript
// ✅ 简洁且安全
const paymentResult = await this.paymentService.createPayment({
  orderNo: checkoutPayDto.v,
  amount: orderInfo.amount,
  currency: orderInfo.currency,
  paymentMethod: 'CREDIT_CARD',  // 只需指定支付方式
  userId: '',
  cardNumber: checkoutPayDto.number,
  cardExpiry: checkoutPayDto.expiry,
  cardCvv: checkoutPayDto.verification_value,
  cardHolderName: checkoutPayDto.name,
  // ✅ 不再需要传入 channelCode 和 channelNotify
  // PaymentService 会自动选择最合适的通道并附加配置
});
```

## 🔄 自动化流程

```
调用方传入
{
  orderNo, amount, currency,
  paymentMethod: 'CREDIT_CARD'
}
    ↓
PaymentService.createPayment()
    ↓
1. 根据 paymentMethod 筛选支持的通道
   ├─ paymentMethod === 'CREDIT_CARD'
   └─ 支持的方式包含 'CREDIT_CARD' 的通道
    ↓
2. 根据 currency 过滤通道
   ├─ currency === 'USD'
   └─ supportedCurrencies 包含 'USD' 的通道
    ↓
3. 根据 amount 过滤通道
   ├─ amount >= minAmount
   └─ amount <= maxAmount
    ↓
4. 按 priority 排序（数字越小优先级越高）
    ↓
5. 选择优先级最高的通道
   └─ 例如: X_PAY_CREDIT (priority=1)
    ↓
6. 自动附加通道信息
   ├─ channelCode = 'X_PAY_CREDIT'
   ├─ notifyUrl = 'https://store.fif.com/api/payment/notify/xpay'
   └─ 其他配置...
    ↓
7. 调用对应的支付策略
    ↓
8. 返回结果（包含使用的通道信息）
{
  success: true,
  paymentNo: 'PAY123',
  payUrl: 'https://...',
  channelCode: 'X_PAY_CREDIT',      // ✅ 返回实际使用的通道
  channelName: 'X支付-信用卡'       // ✅ 返回通道名称
}
```

## 🎯 通道选择逻辑

`selectPaymentChannel()` 方法会按照以下规则自动选择通道：

```typescript
async selectPaymentChannel(
  amount: number,
  currency: string,
  paymentMethod: string,
): Promise<PaymentChannelEntity | null> {
  // 1. 查询所有启用的通道
  const channels = await this.paymentChannelRepo.find({
    where: { isActive: true },
    order: { priority: 'ASC' },
  });
  
  // 2. 过滤支持的通道
  const supported = channels.filter(ch => {
    const supportCurrency = ch.supportedCurrencies.includes(currency);
    const supportMethod = ch.supportedMethods.includes(paymentMethod);
    const amountValid = amount >= ch.minAmount && amount <= ch.maxAmount;
    
    return supportCurrency && supportMethod && amountValid;
  });
  
  // 3. 返回优先级最高的（已按 priority ASC 排序）
  return supported[0];
}
```

### 选择示例

| 支付方式 | 货币 | 金额 | 选择的通道 | 原因 |
|---------|------|------|-----------|------|
| CREDIT_CARD | USD | 100 | X_PAY_CREDIT | 优先级1，支持所有方式 |
| MASTERCARD | USD | 100 | X_PAY_CREDIT_MASTERCARD | 专门支持MASTERCARD |
| CREDIT_CARD | IDR | 100 | X_PAY_CREDIT | 只有它支持IDR |
| MASTERCARD | IDR | 100 | X_PAY_CREDIT | Mastercard通道不支持IDR |

const result = await this.paymentService.createPayment({
