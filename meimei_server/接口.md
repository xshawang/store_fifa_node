## 1. 基础接口

### 1.1 健康检查
- **URL**: `GET /`
- **权限**: 公开
- **入参**: 无
- **出参**: 
```json
{
  "code": 200,
  "msg": "Ok",
  "data": {
    "serverTime": 1234567890
  }

```
- **实现逻辑**: 返回服务器当前时间戳，用于健康检查

## 2. 登录认证

### 2.1 获取图片验证码
- **URL**: `GET /captchaImage`
- **权限**: 公开
- **入参**: 无
- **出参**: `ResImageCaptchaDto` - 包含验证码图片和唯一标识
- **实现逻辑**: 生成图片验证码，保存到Redis，返回图片base64和验证码key

### 2.2 用户登录
- **URL**: `POST /login`
- **权限**: 公开
- **入参**: `ReqLoginDto`
  - `userName`: 用户名
  - `password`: 密码
  - `code`: 验证码
  - `uuid`: 验证码key
- **出参**: `ResLoginDto` - 包含access_token
- **实现逻辑**: 
  1. 验证验证码
  2. LocalAuthGuard验证用户名密码
  3. 生成JWT token
  4. 保存用户信息到Redis

### 2.3 获取用户信息
- **URL**: `GET /getInfo`
- **权限**: 已登录用户
- **入参**: 从Token中获取userId
- **出参**: 用户详细信息、角色、权限
- **实现逻辑**: 根据userId查询用户完整信息，包括角色和权限标识

### 2.4 获取用户路由
- **URL**: `GET /getRouters`
- **权限**: 已登录用户
- **入参**: 从Token中获取userId
- **出参**: `Router[]` - 路由配置数组
- **实现逻辑**: 根据用户角色获取菜单权限，构造成前端路由结构

### 2.5 退出登录
- **URL**: `POST /logout`
- **权限**: 公开
- **入参**: Headers.Authorization - token
- **出参**: 无
- **实现逻辑**: 从Redis中删除该token对应的用户信息

## 3. 文件上传

### 3.1 单文件上传
- **URL**: `POST /common/upload`
- **权限**: 已登录用户
- **入参**: 
  - `file`: 文件 (multipart/form-data)
  - `fileName`: 文件名 (query)
- **出参**: 
```json
{
  "fileName": "string",
  "originalname": "string",
  "mimetype": "string"
}
```
- **实现逻辑**: 接收单个文件上传，返回文件信息

### 3.2 多文件上传
- **URL**: `POST /common/uploads`
- **权限**: 已登录用户
- **入参**: `files`: 文件数组 (multipart/form-data)
- **出参**: 文件信息数组
- **实现逻辑**: 接收多个文件上传（暂未完全实现）

## 4. 导航管理

### 4.1 获取导航数据
- **URL**: `GET /nav/data`
- **权限**: 公开
- **入参**: 无
- **出参**: `NavDataItem[]` - 完整导航数据
- **实现逻辑**: 获取所有分类及其下的网站，树形结构返回

### 4.2 分页获取分类列表
- **URL**: `GET /nav/categories`
- **权限**: 已登录用户
- **入参**: `PaginationDto` (pageNum, pageSize)
- **出参**: 分页数据 `{rows: CategoryResponse[], total: number}`
- **实现逻辑**: 分页查询分类列表

### 4.3 获取分类详情
- **URL**: `GET /nav/categories/:id`
- **权限**: 已登录用户
- **入参**: `id` (路径参数)
- **出参**: `CategoryResponse`
- **实现逻辑**: 根据ID查询分类详情

### 4.4 创建分类
- **URL**: `POST /nav/categories`
- **权限**: 已登录用户
- **入参**: `CreateCategoryDto`
- **出参**: `CategoryResponse`

- **入参**:

### 4.6 删除分类

- **权限**: 已登录用户
- **入参**: `id` (路径参数)
- **出参**: `BaseResponse`

## 通用说明

### 响应格式
所有接口统一返回格式：
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```

### 分页响应格式
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {
    "rows": [],
    "total": 0
  }
}
```

### 权限说明
- `@Public()`: 公开接口，无需登录
- `@RequiresPermissions('xxx:xxx:xxx')`: 需要特定权限标识
- 无装饰器: 需要登录但无特定权限要求

### 通用Query参数
- `pageNum`: 页码（默认1）
- `pageSize`: 每页数量（默认10）
